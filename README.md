## 環境構築
1. リポジトリをclone 

- *git clone https://github.com/atsuki-uenishi/RxSwift-Practice1

2. cocoapodsをインストール 

- *sudo gem install cocoapods*

3. cocoapodsをセットアップ

- *pod setup*

4. ライブラリのインストール

- *pod install
